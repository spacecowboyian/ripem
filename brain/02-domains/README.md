# Domains

This directory contains one subdirectory per business domain in ripem.

## Structure

Each domain follows this standard structure:

```
/02-domains/[domain-name]/
  overview.md       — What the domain is and why it exists
  business-rules.md — Constraints and rules that govern the domain
  workflows.md      — Step-by-step user and system workflows
  data-model.md     — Entities, attributes, and relationships
```
