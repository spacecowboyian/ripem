# Shared components

This directory contains documentation for components that are shared across platforms (iOS and web).

Each component subdirectory follows this structure:

```
/[component-name]
  overview.md        — What the component does and when to use it
  api.md             — Props, parameters, and public interface
  usage.md           — Usage examples and guidance
```

> *(Add component subdirectories here as shared components are defined.)*
