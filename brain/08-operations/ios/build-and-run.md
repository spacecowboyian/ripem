# iOS — Build and run

## Prerequisites

- Xcode *(version to be specified)*
- Swift *(version to be specified)*
- Apple Developer account *(for device builds)*

## Steps

1. Open the Xcode project or workspace.
2. Select the appropriate scheme (Development, Staging, or Production).
3. Select a simulator or connected device.
4. Press `Cmd + R` to build and run.

## Schemes

| Scheme | API environment | Notes |
|--------|----------------|-------|
| Development | localhost | For local development |
| Staging | Staging API | For testing against staging |
| Production | Production API | For release builds only |
