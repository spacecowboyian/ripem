# iOS — TestFlight

## Overview

TestFlight is used for distributing pre-release builds to internal and external testers.

## Build upload process

> *(Describe how to archive and upload a build to App Store Connect.)*

1. Select the Production scheme.
2. Set the destination to "Any iOS Device".
3. Product → Archive.
4. In the Organizer, select the archive and click "Distribute App".
5. Choose "App Store Connect" → "Upload".
