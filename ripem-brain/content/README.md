# Content

**Purpose**: Houses sample builds, marketing copy, onboarding narratives, and FAQs for use by agents writing marketing, designing onboarding, or creating examples.

**For Whom**: Agents writing marketing, designing onboarding, or creating examples.

## Overview

This section contains ready-to-use content assets. Marketing copy and onboarding flows should reflect the brand voice in [/naming/brand_guidelines.md](../naming/brand_guidelines.md).

## Key Sections

- [sample_builds.md](./sample_builds.md): Example car builds (fictional, for demos and marketing)
- [marketing_copy.md](./marketing_copy.md): Marketing messages, taglines, and App Store copy
- [onboarding_flows.md](./onboarding_flows.md): Onboarding narratives and copy
- [faq.md](./faq.md): Frequently asked questions

## Quick Reference

- Brand voice: Authentic, direct, car-culture-native
- Tone: Knowledgeable friend, not corporate tool
- See [/naming/brand_guidelines.md](../naming/brand_guidelines.md) for full voice guidelines
