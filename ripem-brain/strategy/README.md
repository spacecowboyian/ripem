# Strategy

**Purpose**: Defines growth mechanics, user acquisition, retention strategies, competitive positioning, and risk mitigation.

**For Whom**: Agents optimizing growth, analyzing competition, or mitigating risks.

## Overview

RiPeM's growth strategy is built on organic virality. The watermark loop is the primary acquisition engine. Community seeding in niche Facebook Groups and influencer partnerships accelerate early growth.

## Quick Reference

- **Primary growth**: Viral watermark loop (organic)
- **Early acquisition**: Niche Facebook Groups + influencer seeding
- **Retention**: Build progression, AI value, community engagement
- **Key risk**: Cold-start problem (content before community)
